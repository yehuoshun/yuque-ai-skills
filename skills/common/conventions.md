# 通用约定

## book_id 格式

`book_id` 参数支持两种格式：

| 格式 | 示例 | 说明 |
|------|------|------|
| 数字 ID | `67048677` | 知识库数字 ID |
| namespace | `yehuoshun/gi49zs` | `{login}/{book_slug}` 格式 |

## 文档标识

| 标识 | 格式 | 使用场景 |
|------|------|------|
| `id` | 数字 | 文档/知识库的唯一 ID |
| `slug` | 字符串 | 文档路径，配合 `book_id` 使用 |

> ⚠️ `slug` 不能直接当 `id` 传——`repos/docs/:slug` 会 404，需要 `repos/:book_id/docs/:slug`。

## 分页

| 参数 | 说明 |
|------|------|
| `offset` | 偏移量，默认 0 |
| `limit` | 每页数量，≤100，默认 100。超过 100 返回 422 |

## 输出裁剪

所有工具默认返回精简字段（去除 `_serializer`、`type` 等元数据）。传 `raw=true` 可获取全量原始 JSON。

## 频率限制

- API v2：每小时 5000 次 / 每秒 100 次
- Cookie 态接口：QPS 未知，保守单次请求，不要并发
- 429 时重试 1s→2s→3s，最多 3 次
